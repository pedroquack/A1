Documentação atividade A1 Desenvolvimento Web

- O sistema possui duas paginas html, uma de instruções de utilização e outra para a funcionalidade do site, com toda a parte de gerenciamento de tarefas
- O sistema possui apenas um formulário que é modificado pelo JS dependendo da situação, por padrão ele é um formulário de criação de tarefas, exceto se o input escondido no formulario (id-tarefa) esteja com algum valor, se houver, ele vai ser um formulário para modificar a tarefa correspondente a esse id
- As tarefas são listadas novamente sempre que ocorre alguma função de modificação ou inserção das tarefas
- As tarefas são armazenadas no localStorage, como um vetor de objetos
